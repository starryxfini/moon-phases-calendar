# moon-phases-calendar

this is a little *CLI* moon phase calendar, you run the file and via api you'll get the ascii moon phase of the date you entered.

yes the code isnt perfect but it works and im proud of it :D

## how it works:
- download the .py file
- go to the [API](https://docs.astronomyapi.com/endpoints/studio/moon-phase) and create yourself an account
- insert your key in the code
- insert your coordinates in the code
- go ahead and run it via the terminal :D

API: https://docs.astronomyapi.com/endpoints/studio/moon-phase
